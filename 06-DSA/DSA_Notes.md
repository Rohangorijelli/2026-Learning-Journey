* What is intstream?
This is Part of Java 8 streams API. specialized stream for working with primitive int values. Belongs to the package: java.util.stream. IntStream is an interface in java.util.stream.
Actual work is done by classes like Streams, StreamSupport, and specialized implementations (IntPipeline, ReferencePipeline).
